# Dossier des documents techniques

Ce dossier contient les documents techniques liés aux produits du stock SEMPA :

- Fiches techniques (PDF)
- Photos de produits
- Documentation fournisseurs
- Autres médias

## Utilisation

Les fichiers stockés ici sont accessibles depuis :
- La vue "Rapports" de StockPilot
- Les fiches produits individuelles

## Format acceptés

- PDF
- Images (JPG, PNG, GIF)
- Documents Office (DOC, XLS)
